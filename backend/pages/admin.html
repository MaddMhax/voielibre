Aide

  1. Rendez-vous sur le portail développeurs SNCF : numerique.sncf.com/startup/api.
  2. Créez un compte gratuit (« Demander un token ») et validez votre adresse e-mail.
  3. Vous recevez votre clé API (un long jeton alphanumérique) par e-mail et/ou dans votre espace développeur.
  4. Renseignez-la dans la variable SNCF_API_KEY du docker-compose.yml (ou d'un fichier .env), puis relancez l'application (docker compose up -d --build).
  5. L'offre gratuite couvre 5 000 requêtes/jour et 150 000/mois — suivez votre consommation dans le panneau « Quota API SNCF » ci-dessus.

La clé sert d'identifiant HTTP Basic (mot de passe vide) ; l'application gère cette authentification automatiquement. Les voies de départ proviennent d'un flux open data séparé qui ne nécessite aucune clé.
